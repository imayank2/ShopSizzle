# ShopSizzle
ShopSizzle is a user-friendly e-commerce platform where users can explore products, add them to their cart, and proceed to purchase.
Built with a robust backend in Spring Boot using the MVC (Model-View-Controller) architecture and a dynamic frontend in React, ShopSizzle provides essential e-commerce features:

    Add Products: Users can browse and add items to the store.
    Update Quantities: Easily modify product quantities to manage stock.
    Delete Products: Remove products from the catalog with ease.
    Shopping Cart: A seamless add-to-cart experience, with a final checkout option.
    Buy Option: Complete purchases through a streamlined buying process.

Ideal for anyone looking to implement or extend e-commerce functionality, ShopSizzle combines powerful backend logic with a smooth, React-based frontend.
